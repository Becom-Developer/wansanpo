# NAME

/auth - wansanpo アプリ認証

# SYNOPSIS

## URL

- GET - `/auth/entry` - ユーザー登録画面
- POST - `/auth/entry` - ユーザー登録実行
- GET - `/auth/login` - ログイン画面
- POST - `/auth/login` - ログイン実行
- POST - `/auth/logout` - ログアウト実行
